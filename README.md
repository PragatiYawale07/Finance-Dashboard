Project Name: Finance Dashboard
Project Overview:
Finance Dashboard is a Spring Boot and Java backend project that allows users to manage financial records, users, and roles. 
It uses PostgreSQL as the database and includes role-based access with Spring Security.

Features:

User management where Admin can manage users
Role-based access: Admin, Analyst, Viewer
Financial records management: add, update, view
APIs secured with HTTP Basic Authentication

Technologies Used:
Java 17, Spring Boot 3, Spring Security, Spring Data JPA, PostgreSQL, Maven, Poastman for Api Test

Running the Project:

Clone the repository from https://github.com/PragatiYawale07/Finance-Dashboard

Import the project into IntelliJ IDEA 
Update application.properties with your PostgreSQL credentials
Run the project
APIs will be available at http://localhost:8080/api/


API Documentation:
You can use the Postman collection file Finance-Dashboard-Api.postman_collection.json to test all APIs
